Hoe kan ik me best beschermen tegen hacken?

1. Kies een sterk paswoord.

Voetbal123, Tino123, Tennis123, allemaal voorbeelden van slechte paswoorden... dus hoe maak je nu echt een goed paswoord?


Je pakt best een heel "random" woord, iets dat zomaar in je opkomt, bv. appelkaassaus. en die woorden laat je volgen door een zelfverzonnen cijfer bv. 156723. Deze twee samen maken een sterk paswoord.


2. Zet 2 factor authenticatie aan.

Bij 2 stappen authenticatie zenden de servers van bv. Google, Facebook, etc. een SMS bericht met een code naar je GSM wanneer je aanlogt op een van hun services. 3. Vertrouw niemand met jouw account.

Veel mensen delen hun wachtwoord met vrienden of vriendinnen. Doe dit alstublieft niet! Er is geen enkele reden waarom zij toegang zouden moeten hebben tot jouw account. Veel redenen waarom rare of gemene berichten worden verzonden vanaf een "gehackt" account is omdat een vriendin of een vriend ruzie heeft met de "gehackte" vriend/vriendin en daarom zo schade aanricht.


4. Klik geen rare links open.

Een rare SMS van zogezegd bpost, de overheid, etc. iedereen kent het wel. Klik deze links niet open. De bank, de overheid, bpost zal nooit via SMS vragen om een betaling in orde te brengen. Je kan deze berichten best gewoon verwijderen. Accounts op instagram dat je rare links sturen kan je ook best blokkeren. Je klikt best nooit zomaar een link open, controleer deze best altijd eerst eens door te googlen als je twijfelt.


5. Gebruik altijd secure protcollen.

Bij het gebruik van HTTP of FTP zendt je computer en de externe server de bestanden gewoon door in leesbare tekst. Dit betekend dat als er een hacker fysiek tussen de verbinding van de server en jou zit, de hacker al de bestanden kan zien.

Als oplossing hiervoor bestaat HTTPS en FTPS, zolang je deze gebruikt ben je veilig, zelfs op openbare WiFi.
